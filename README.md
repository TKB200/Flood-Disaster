# Flood Disaster Citizen Data Collection Portal

This is a premium, responsive web application designed to collect data from Sri Lankan citizens affected by flood disasters.

## Features
- **Premium UI/UX**: Modern glassmorphism design with smooth animations.
- **Responsive**: Works perfectly on mobile, tablet, and desktop.
- **Dynamic Form**: District dropdown automatically updates based on the selected Province.
- **Validation**: Includes Sri Lankan NIC validation and required field checks.

## How to Use
1. Simply open the `index.html` file in any modern web browser (Chrome, Edge, Firefox).
2. Fill in the details.
3. Click "Submit Information".

## Files
- `index.html`: The main application structure.
- `style.css`: Custom styling and animations.
- `script.js`: Form logic and validation.

## Technical Details
- Built with HTML5, CSS3 (Variables, Flexbox, Grid), and Vanilla JavaScript.
- No external dependencies required (Fonts are loaded from Google Fonts).
